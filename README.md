# Autofill Search Bar Project

## Overview
This is a simple autofill search bar that suggests fruit names as the user types. It filters the list of fruits based on user input, displays matching results in a dropdown, and allows the user to select a suggestion to populate the search bar.

## Features
- Search bar with placeholder text ("Search fruit üçé").
- Case-insensitive filtering of fruit names.
- Dropdown shows suggestions based on user input.
- Highlighting of suggestions on hover.
- Populate search bar with the selected suggestion.

## Files
- **index.html**: Contains the structure of the search bar and dropdown.
- **style.css**: Styles the page, search bar, and dropdown.
- **script.js**: Contains the JavaScript logic for filtering results, displaying the dropdown, and handling user interactions.

## Setup
1. Clone or download the project.
2. Open `index.html` in your browser to view the search bar in action.

## Usage
- Type into the search bar to see suggestions for fruit names.
- Hover over suggestions to highlight them.
- Click on a suggestion to populate the search bar.
